    Option Récursive

    English | Français | 한국어 | 日本語

    Si vous souhaitez convertir tous les fichiers de base du répertoire actuel et des sous-répertoires, utilisez l'option --recursive ou -r.

    mmg --recursive

    Soyons par exemple que le chemin actuel est le suivant.

    .
├─ docs/
│  └─ Retriever.base.md
├─ jupyter/
│  └─ Collie.base.ipynb
└─ Corgi.base.md

    En utilisant l'option récursive, vous pouvez convertir tous les fichiers de base du répertoire actuel et des sous-répertoires.

    $ mmg -r
----------------------
 ✅ Corgi.base.md
 ✅ docs\Retriever.base.md
 ✅ jupyter\Collie.base.ipynb
----------------------
 => 3 base files were found.
    Do you want to convert these files? [y/N]

    Vous pouvez également utiliser l'option récursive lorsque vous passez un fichier en argument, et vous pouvez également l'utiliser avec l'option --output. (A propos de l'option --output, voir ici.) Cependant, il ne peut pas être utilisé avec l'option traitement par lots. Si vous utilisez --recursive et --batch ensemble, --batch est ignoré, vous ne pouvez donc pas les utiliser ensemble.
